Introduction

An array is a data type provided by javaScript specifically for storing sequences of values. It is written as a list of values between square brackets, separated by commas.The notation for getting at the elements inside an array also uses square brackets. A pair of square brackets immediately after an expression, with another expression inside of them, will look up the element in the left-hand expression that corresponds to the index given by the expression in the brackets.

The first index of an array is zero, not one. So the first element is retrieved with [0]. Zero-based counting has a long tradition in technology and in certain ways makes a lot of sense, but it takes some getting used to. Think of the index as the amount of items to skip, counting from the start of the array.

    Key array concepts:
  • The elements in an array are stored as the array’s properties, using numbers as property names.
  • Those elements are accessed via bracket notation
  • The length property of an array tells us how many elements it has.
  • An array unlike strings are mutable i.e they can be changed for convenience.

Looping through an Array

let example = [1, 45, 'one', 'two', 14]; for(let i = 0; i < example.length; i++) { let entry = example[i]; // Do something with entry }

This kind of loop is common in classical JavaScript—going over arrays one element at a time is something that comes up a lot, and to do that you’d run a counter over the length of the array and pick out each element in turn. There is a simpler way to write such loops in modern JavaScript. When a for loop looks like this, with the word of after a variable definition, it will loop over the elements of the value given after of. This works not only for arrays but also for strings and some other data structures.

let example = [1, 45, 'one', 'two', 14]; for (exam of example) { // Do something with exam } // or example.forEach({ //loop through example });

pop and push

let example = [1, 45, 'one', 'two', 14]; example.push(15); console.log(example); // returns [1, 45, 'one', 'two', 14, 15]; console.log(example.pop()); // returns [1, 45, 'one', 'two', 14];

The push method adds values to the end of an array, and the pop method does the opposite, removing the last value in the array and returning it. This methods mutates the array i.e it changes the array. In functional programming, its common practice to make a copy of an array for any intended mutation.

shift and unshift

The shift and unshift methods works basically like pop and push with the key difference being that shift adds a value to the begining of the array and unshift deletes from the begining of the array.

slice and splice

Another fundamental array method is slice, which takes start and end indices and returns an array that has only the elements between them.

    Parameters
  • The start index is inclusive.
  • The end index exclusive.
  • When the end index is not given, slice will take all of the elements after the start index.
  • You can also omit the start index to copy the entire array.

Slice does not mutate the array, it makes a copy of the array and returns that copy. Splice on the otherhand mutates the array and deletes all values of the array not covered in the index given.

let example = [1, 45, 'one', 'two', 14]; let one = example.slice(1); console.log(one); // returns [45, 'one', 'two', 14]; console.log(example); // returns [1, 45, 'one', 'two', 14]; // original array does not change example.splice(1); console.log(example); // returns [45, 'one', 'two', 14]; // original array changes.

includes and indexOf

Arrays have an includes method that checks whether a given value exists in the array. It returns true if the value is included in the array and false if it's not included.

The indexOf method is a powerful array method for searching for values in an array. The indexOf method returns the index of the value given as an argument if it's in the array and it returns -1 if its not found in the array.

map and filter

The map() method creates a new array populated with the results of calling a provided function on every element in the array. The filter() method creates a new array with all elements that pass the test implemented by the provided function.

The map() and filter() method takes three arguments

    Parameters
  • CurrentValue: The current element being processed
  • Index: The index of the current element being processed in the array
  • The array map() or filter() was called upon

let example = [1, 3, 5, 7, 10]; console.log(example.map(e => e * 2)); // returns [2, 6, 10, 14, 20]; console.log(example.filter(e => e >= 5)); // returns [5, 7, 10];

reduce

A common thing to do with arrays is to compute a single value from them. The high-order operation that represents this pattern is called reduce It builds a value by repeatedly taking a single element from the array and combining it with the current value. When summing numbers, you’d start with the number zero and, for each element, add that to the sum.

The parameters for reduce() are

    Parameters
  • A combining function
  • A start value usually Zero
  • The array reduce() was called upon
  • If your array contains at least one element, you are allowed to leave off the start argument. The method will take the first element of the array as its start value and start reducing at the second element.

let example = [1, 3, 5, 7, 10]; console.log(example.reduce((a, b) => { a + b; }); // returns [26]; console.log(example.reduce((a, b) => { a + b; }, 5); // returns [31]; with 5 as the start value

every and some

The every() method is called on an array to see if every value in the array passes a defined test. it returns a boolean value. The some() method checks to see if at least one of the values of the array passes the set test. it returns a boolean as the result.

let example = [1, 3, 5, 7, 10]; console.log(example.every(s => s > 4); // returns false; console.log(example.some(s => s > 4); // returns true;

toString and Array.from

The toString() method converts all the elements within an array into a string. The Array.from() static method generates an array from an array-like or iterable object.

The parameters for Array.from() are

    Parameters
  • An array like object to convert to an array
  • A map function to call on every element of the array

let example = [1, 3, 5, 7, 10]; console.log(example.reduce((a, b) => { a + b; }); // returns [26]; console.log(Array.from('Array')); // returns ['A', 'r', 'r', 'a', 'y']; console.log(Array.from(example, x => x + x)); // returns [2, 6, 10, 14, 20];
